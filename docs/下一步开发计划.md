## 🚀 立即开始的任务

### 1. 安装后端依赖
```bash
cd backend
npm install
```

### 2. 配置环境变量
创建 `.env` 文件：
```env
DATABASE_URL="postgresql://username:password@localhost:5432/zapdb"
JWT_SECRET="your-secret-key"
```

### 3. 初始化数据库
```bash
npx prisma generate
npx prisma db push
```

### 4. 启动后端服务
```bash
npm run start:dev
```

### 5. 创建前端认证页面
- 登录页面
- 注册页面
- 认证状态管理

## 🔧 技术栈

### 后端
- **框架**: NestJS + TypeScript
- **数据库**: PostgreSQL + Prisma
- **认证**: JWT + Passport
- **验证**: class-validator

### 前端
- **框架**: React 18 + TypeScript
- **UI库**: Ant Design 5.x
- **状态管理**: Zustand
- **路由**: React Router
